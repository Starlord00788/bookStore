# 📚 E-Commerce Bookstore Website

## 🚀 Overview
The **E-Commerce Bookstore Website** is a full-stack web application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. This platform offers a seamless online book-shopping experience with features like cart management, order tracking, and secure payments via **Razorpay**. **Cloudinary** integration ensures optimized book cover storage for fast loading, while **Tailwind CSS** and **Chakra UI** provide a modern, responsive design.

## 🎯 Features
- 🛍️ **Comprehensive E-Commerce Functionality** – Browse, add to cart, and place orders seamlessly.
- 💳 **Secure Payment Integration** – Process transactions safely using Razorpay.
- 📚 **Cloudinary Integration** – Optimized and fast-loading book cover images.
- 🔍 **Advanced Search & Filtering** – Find books quickly based on categories, authors, and price range.
- 🛒 **Cart & Order Management** – Smooth checkout experience with real-time order tracking.
- 🎨 **Modern UI/UX** – Built with React.js, Tailwind CSS, and Chakra UI for an intuitive interface.
- ⚡ **Optimized Performance** – API optimizations and caching mechanisms ensure smooth user experience.

## 🛠️ Technologies Used
- **Frontend** – React.js, Tailwind CSS, Chakra UI
- **Backend** – Node.js, Express.js
- **Database** – MongoDB
- **Cloud Storage** – Cloudinary
- **Payment Gateway** – Razorpay
- **Authentication** – JWT-based secure login
- **Caching & Performance** – API optimization techniques

## 📦 Installation & Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/Starlord00788/bookStore.git
   ```
2. Navigate to the project folder:
   ```bash
   cd bookstore
   ```
3. Install dependencies for both frontend and backend:
   ```bash
   npm install
   cd client && npm install
   ```
4. Set up environment variables:
   - Create a `.env` file in the root directory and add MongoDB, Cloudinary, and Razorpay credentials.
5. Start the backend server:
   ```bash
   npm run dev
   ```
6. Start the frontend server:
   ```bash
   cd client
   npm run dev
   ```
7. Open your browser and visit:
   ```
   http://localhost:3000
   ```
8. Start browsing and buying your favorite books! 📖
